Introduction to Web Developnment

Web Developnment is the process of Creating Websites and Application that run on Internet

Objectives of WebDevelopnment

  • Understand Basic Structure Of Html
  • Learn From formating Text Alignment
  • Use list,images and HyperLinks

Core Components of a WebPage

  1. HTML
    • Define Structure of Page
    • Use tags Like <html>, <head>, <body>
  2. CSS
    • Used for Styling Web Pages
    • Control Layouts And colours
  3. JavaScript
    • Adds Interactivity
    • Handles events and validation

Text Formatting Options

HTML allows text to be bold, italic, underlined, and even coloured to make the content attractive

Special Character

Some special symbol used in HTML

  • Copyright Symbol: ©
  • Registered Trademark: ®
  • Trademark: ™
  • Less Than Sign: <
  • Greater Than Sign: >
  • Ampersand: &
